可藉由剖析對應到公用 API 的原生程式庫的標頭檔來運作[繫結定義](~/cross-platform/macios/binding/binding-types-reference.md)（否則手動完成此程序）。 目標 Sharpie 不會單獨使用時，建立繫結，但它可以幫助您開始使用 ！
