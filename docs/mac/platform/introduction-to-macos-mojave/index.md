## <a name="core-ml-2-natural-language-framework"></a>核心 ML 2： 自然語言架構

該自然語言架構中，組件的核心 ML 2，可讓應用程式，以執行各種類型的語言分析。 比方說，它可以用來識別組件的語音，並判斷由文字區塊的語言。
