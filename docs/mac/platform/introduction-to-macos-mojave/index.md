## <a name="core-ml-2-natural-language-framework"></a>核心 ML 2： 自然語言架構

該自然語言架構中，部分核心 ML 2，可讓執行各種類型的語言分析的應用程式。 例如，它可以用來識別部分，並判斷由文字區塊的語言。
