# <a name="synchronizing-data-with-web-services"></a>同步處理資料與 Web 服務

_離線同步處理可讓使用者互動與行動應用程式、 檢視、 加入或修改資料，即使在沒有網路連線。變更會儲存在本機資料庫中，並變更裝置在線上時，一旦可以同步處理與 web 服務。_
