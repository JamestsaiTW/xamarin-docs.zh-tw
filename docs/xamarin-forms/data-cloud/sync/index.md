# <a name="synchronizing-data-with-web-services"></a>與 Web 服務同步資料

_離線同步處理可讓使用者互動與行動應用程式、 檢視、 加入或修改資料，即使在沒有網路連線。變更會儲存在本機資料庫中，並變更該裝置上線後，可以同步處理與 web 服務。_
