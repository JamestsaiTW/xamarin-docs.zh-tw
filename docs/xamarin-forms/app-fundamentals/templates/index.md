# <a name="templates"></a>範本

## <a name="control-templatescontrol-templatesindexmd"></a>[控制項範本](control-templates/index.md)

Xamarin.Forms 控制項範本提供輕鬆佈景主題和 re 佈景主題的功能在執行階段應用程式頁面。
