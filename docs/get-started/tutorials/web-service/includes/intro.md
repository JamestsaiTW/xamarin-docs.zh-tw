在此教學課程中，您將了解如何：

> [!div class="checklist"]
> - 使用 NuGet 套件管理員將 Newtonsoft.Json 新增至 Xamarin.Forms 專案。
> - 建立 Web 服務類別。
> - 取用 Web 服務類別。
