# <a name="f35"></a>F&#35;

_F # 是強型別功能的程式設計語言設計.NET 上執行_

F # 結合 succinctness、 表現和撰寫性型別推斷功能性程式設計，強型別程式碼的編譯時期安全和執行階段支援、 程式庫、 互通性、 工具和.NET 物件模型。

F # 是現在可以使用 「 現成的 」 做為所有 Xamarin 支援的平台上的 Xamarin 使用者的程式設計語言： Android、 OS X 和 iOS。

- [開始使用 F #](overview.md)
- [F# 範例](samples.md)
